# Rize Query Patterns

## Goal
Describe reusable request patterns for the Rize skill without relying on one person’s private usage habits.

## Common Query Shapes
- Current week summary
- Current month summary
- Custom date range summary
- Category-inclusive breakdown when explicitly requested
- Save or reuse the structured output in another workflow

## Public Guidance
- Keep defaults explicit in the script or wrapper that calls the API.
- Describe date-range parsing and category toggles generically.
- Avoid embedding private file targets or personal productivity habits in public examples.
